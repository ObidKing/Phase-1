##Problem statement

This is a simple calculator to do addition, subtraction, multiplication and division

## User story
A user should be able to add two numbers

A user should be able to divide two numbers

A user should be able to multiply two numbers

A user should be able to subtract two numbers



## Functional Requirement

The application should be able to accept additional, subtractions, multiplications and divisions of two variable 


##Rule and assumptions

we should be able to do basic add, sub, multiply and divide

assume we only accepting integer for now
